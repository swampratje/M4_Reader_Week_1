Subsequences and Bolzano–Weierstrass

NoteSubsequence

Given a sequence \(\{a_n\}_{n\in \mathbb{N}}\) and an increasing function \(f:\mathbb{N}\rightarrow \mathbb{N}\), the sequence \[ \{a_{f(n)}\}_{n\in \mathbb{N}} \] is called a subsequence of \(\{a_n\}\).

NoteExample: a constant subsequence

Let \(a_n=(-1)^n\). Then \(f(n)=2n\) defines the constant subsequence \(b_n=a_{f(n)}=1\).

Theorem 1 Theorem (Bolzano–Weierstrass). Let \(\{x_n\}\) be a bounded sequence in \(\mathbb{R}^m\). Then it has a convergent subsequence.

Proof (sketch)

For \(m=1\): put all points in an interval \([a_0,b_0]\). Bisect repeatedly and keep a subinterval that contains infinitely many points. This yields nested intervals with lengths going to zero. Pick one sequence element from each interval with increasing indices; by squeezing, the subsequence converges.

For general \(m\): apply the one-dimensional result to coordinates iteratively and use a diagonal subsequence so that all coordinates converge.

NoteExample: bounded but not convergent; has convergent subsequence

The sequence \((-1)^n\) does not converge, but it is bounded and has convergent subsequences, e.g. \(a_{2n}=1\).

NoteExample: \((\cos n,\sin n)\) is bounded; has a convergent subsequence

The sequence \(a_n=(\cos n,\sin n)\) in \(\mathbb{R}^2\) does not converge. But since \(\Vert a_n\Vert=1\) for all \(n\), it is bounded and therefore has a convergent subsequence by Bolzano–Weierstrass.

1 Subsequences

1.1 Definition

A subsequence is obtained by selecting terms from a sequence while preserving their order.

Let \(\{a_n\}_{n\in\mathbb{N}}\) be a sequence and let \(f:\mathbb{N}\to\mathbb{N}\) be a strictly increasing function.
Then the sequence \(\{a_{f(n)}\}_{n\in\mathbb{N}}\) is called a subsequence of \(\{a_n\}_{n\in\mathbb{N}}\).

Intuitively, a subsequence is “drawn” from the original sequence by keeping some terms and discarding others, but never changing the order.

NoteExample

Let \(a_n = (-1)^n\). If we choose the increasing index map \(f(n)=2n\), then [ b_n := a_{f(n)} = a_{2n} = 1 n, ] so \(\{b_n\}\) is a constant (and hence convergent) subsequence.

2 Bolzano–Weierstrass

Theorem 2 Theorem (Bolzano–Weierstrass).
Let \(\{x_n\}\) be a bounded sequence in \(\mathbb{R}^m\). Then it has a convergent subsequence.

Proof

Step 1: the case \(m=1\).
Let \(\{x_n\}\) be bounded in \(\mathbb{R}\), so there exists \(M>0\) such that [ x_n :=[-M,M] n. ] Bisect \([a_0,b_0]\) into two intervals of equal length. At least one of these subintervals contains infinitely many points of the sequence. Call that interval \([a_1,b_1]\).

Repeat: bisect \([a_1,b_1]\), choose a half-interval \([a_2,b_2]\) containing infinitely many points of the sequence, and continue inductively. This gives nested intervals [ [a_0,b_0] ] with lengths [ b_k-a_k = . ] Moreover, \(\{a_k\}\) is nondecreasing and bounded above, and \(\{b_k\}\) is nonincreasing and bounded below, hence both converge (Monotone Convergence Theorem). Because \(b_k-a_k\to 0\), both converge to the same limit \(\ell\).

Now choose indices inductively: pick \(x_{n_1}\in [a_1,b_1]\), then pick \(x_{n_2}\) with \(n_2>n_1\) and \(x_{n_2}\in [a_2,b_2]\), etc. Then [ a_k x_{n_k}b_k k, ] so by the squeeze theorem \(x_{n_k}\to \ell\).

Step 2: the general case \(m\ge 2\).
Write \(x_n=(x_{n,1},\dots,x_{n,m})\). If \(\{x_n\}\) is bounded, then each coordinate sequence \(\{x_{n,j}\}\) is bounded in \(\mathbb{R}\).

By the one-dimensional case, the first-coordinate sequence \(\{x_{n,1}\}\) has a convergent subsequence. Restrict to the corresponding subsequence of vectors; within that subsequence, the second-coordinate sequence is still bounded, so it has a convergent subsequence. Continue in this way (a diagonal/subsequence extraction argument) until all \(m\) coordinates converge along the same subsequence.

Hence \(\{x_n\}\) has a subsequence whose coordinates converge, so the subsequence converges in \(\mathbb{R}^m\).

NoteExample

The sequence \(a_n = (-1)^n\) does not converge, but it is bounded since \(|a_n|=1\) for all \(n\).
The subsequence \(a_{2n}=1\) converges (to \(1\)).

NoteExample

Consider \(a_n=(\cos n,\sin n)\in\mathbb{R}^2\). This sequence does not converge, but it is bounded since [ |a_n|==1 n. ] Therefore, by Bolzano–Weierstrass, it has a convergent subsequence.
